## 介绍

HyperOS 的轻量级模块，专注于提升日常使用体验。  
基于小米 15 Pro 的最新稳定版 HyperOS China 开发。  

加入 [Telegram 频道](https://t.me/lackluster_stuff) 以反馈问题并获取内部更新。
